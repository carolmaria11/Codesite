    • Description

    Are you curious to know how compatible you are with your crush, partner, or someone you're secretly admiring? Look no further than the Love Calculator! Our fun and intriguing app is designed to help you uncover the mysterious world of romantic compatibility.

    Key Features

    To work out the love score between two people:

    1. -> User Input:
      The code prompts the user to input the names of two people.

    2. -> Name Normalization:
      The input names are converted to lowercase using the lower() method. This ensures that the calculations are case-insensitive.

    3. -> Calculating the Love Score:
      • The code counts the occurrences of the letters 't', 'r', 'u', 'e', 'l', 'o', 'v', and 'e' in the combined names.
      • It then calculates the love score by concatenating the counts and converting them to an integer. For example, if 'true' appears 3 times and 'love' appears 5 times, the love score would be 35.

    4. -> Love Score Interpretation:
      • Based on the love score, the code provides one of three possible messages:
      • If the love score is less than 10 or greater than 90, it suggests "Not a good match."
      • If the love score is between 40 and 60 (inclusive), it suggests "You have a decent chance."
      • Otherwise, it suggests "You are a great match!"

    5. -> Main Function::
      • The if __name__ == "__main__" block ensures that the love_calculator() function is executed when the script is run as the main program.

    6. -> Random Module Import:
      • Although the random module is imported at the beginning of the script, it is not used in the current version of the code. It may have been intended for future enhancements or expansion of the Love Calculator.

    Content used

    1. -> The lower() : function changes all the letters in a string to lowercase.
    2. -> The count() : function will give you the number of times a letter occurs in a string
