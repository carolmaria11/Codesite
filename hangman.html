    • Description

    Hangman is an engaging and classic word-guessing game that will test your vocabulary and deductive skills. Challenge yourself or compete with friends to unravel the hidden word one letter at a time while avoiding the hangman's noose. With a diverse word bank and various difficulty levels, Hangman provides endless entertainment for word enthusiasts of all ages.

    Key Features

    1. -> The Hangman program randomly selects a secret word from a list of secret words.
    2. -> The random module will provide this ability, so line 1 in the program imports it.
    3. -> The Game: Here, a random word is picked up from our collection and the player gets limited chances to win the game.
    4. -> When a letter in that word is guessed correctly, that letter's position in the word is made visible. In this way, all letters of the word are to be guessed before all the chances are over.
    5. -> For convenience, we have given the length of the word + 2 chances. For example, if the word to be guessed is mango, then the user gets 5 + 2 = 7 chances, as mango is a five-letter word.
