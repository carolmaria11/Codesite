    • Description

    This Python program creates a simple weather app with a graphical user interface (GUI) using the tkinter library. Users can enter the name of a city and click a "Search" button to retrieve and display weather information for that city. When you run this script, a window will appear with an entry field to input the city name. After entering a city name and clicking the "Search" button, the weather information for that city will be displayed in the labels, including the location, weather icon, temperature, and description. The app uses the OpenWeatherMap API to fetch real-time weather data based on the city entered by the user.

    Key Features

    1. -> GUI Interface:
      • The program uses tkinter to create a user-friendly interface with various widgets, including an input field for the city name, a search button, and labels for displaying weather information.

    2. -> Weather Data:
      • It fetches weather data from the OpenWeatherMap API based on the user's input. This data includes the location (city and country), a weather icon, temperature in Celsius, and a weather description.

    3. -> Error Handling:
      • The program handles errors gracefully, such as when the entered city is not found, by displaying an error message.

    4. -> Display:
      • The retrieved weather data is displayed dynamically on the GUI, updating labels and displaying the weather icon as an image

    5. -> Styling:
      • The GUI is styled using the ttkbootstrap library to enhance the app's visual appeal.

    Content used

    1. -> Import necessary libraries:
      • tkinter :- for creating the GUI.
      • Requests :- for making HTTP requests to the OpenWeatherMap API.
      • Messagebox :- for displaying error messages.
      • Messagebox :- for displaying error messages.
      • ttkbootstrap :- for GUI styling.

    2. -> Define the get_weather function:
      • This function takes a city name as input and uses the OpenWeatherMap API to retrieve weather information for that city.

    3. -> Create GUI widgets:
      • city_entry: An entry widget where users can input the city name.
      • search_button: A button that triggers the weather search when clicked.
      • location_label: A label to display the location (city and country).
      • icon_label: A label to display the weather icon.

    4. -> Configure widget styles:
      • Styling for the widgets is specified using ttkbootstrap styles for the entry and
      • button widgets. Fonts and sizes are also specified for some labels.

    5. -> Enter the main event loop:
      • The root.mainloop() function is called to start the Tkinter event loop, which handles user interactions and updates the GUI.
