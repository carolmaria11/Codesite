    • Description

    This Python script uses the PIL (Python Imaging Library) to open and compress an image file selected by the user through a file dialog image selected by the user, resize it using the Lanczos resampling algorithm, and save the resized image with a ".jpg" extension with a user-defined file name and location.

    Key Features

    1. -> User Interaction:
      • The program interacts with the user by opening a file dialog to allow them to select an image file for processing.
    2. -> Image Opening:
      • It opens the selected image file using the PIL (Python Imaging Library) and stores the image data in memory for further processing.
    3. -> Image Resizing:
      • The program resizes the opened image using the Lanczos resampling algorithm (PIL.Image.LANCZOS). The dimensions of the resized image are kept the same as the original image, effectively not changing its size.
    4. -> File Saving:
      • The program appends the ".jpg" file extension to the chosen file name before saving.
    5. -> File Dialogs:
      • The code employs the tkinter.filedialog module to create file dialogs for both opening and saving files, enhancing user-friendliness.

    Content used

    1. -> Import Statements:
      • import PIL: Imports the PIL module.
      • From PIL import Image: Imports the Image class from the PIL module for image manipulation.
      • from tkinter.filedialog import *: Imports all symbols from the tkinter.filedialog module, which is used for file dialogs.
    2. -> Function Calls:
      • askopenfilename(): Opens a file dialog for the user to select an image file and returns the selected file's path.
      • PIL.Image.open(file_path): Opens an image file specified by file_path using PIL's Image.open() method.
    3. -> Image Resizing:
      • img.resize((myWidth, myHeight), PIL.Image.LANCZOS): Resizes the opened image using the Lanczos resampling algorithm. The resized image is assigned back to the img variable.
      • Image Saving:
    4. -> img.save(save_path + ".jpg"):
      • Saves the resized image in JPEG format with a ".jpg" file extension using the img.save() method.
