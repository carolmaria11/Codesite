    • Description

    This data analysis project that involves importing, cleaning, and visualizing data from a CSV file named "Diwali Sales Data.csv." The project aims to explore and analyze the sales data, with a focus on different aspects such as gender, age group, state, marital status, occupation, and product categories. It uses popular data manipulation and visualization libraries such as NumPy, pandas, Matplotlib, and Seaborn.

    Key Features

    1. -> Data Import:
      • The code starts by importing necessary Python libraries and loading the CSV file, "Diwali Sales Data.csv," into a pandas DataFrame for further analysis.

    2. -> Data Cleaning:
      • It involves several data cleaning steps, including:
      • Dropping unrelated or blank columns (Status and unnamed1).
      • Checking for and dropping rows with null values using pd.isnull() and df.dropna().
      • Changing the data type of the 'Amount' column to integers using df['Amount'].astype('int').

    3. -> Data Exploration:
      • The code explores the dataset by:
      • Renaming a column from 'Marital_Status' to 'Shaadi' (although this change is not persisted in the DataFrame).
      • Using df.describe() to generate summary statistics for the entire dataset.
      • Using df[['Age', 'Orders', 'Amount']].describe() to generate specific statistics for selected columns.

    4. -> Data Visualization:
      • The code utilizes various visualization techniques to gain insights from the data, such as:
      • Creating a bar chart for gender distribution using Seaborn (sns.countplot()).
      • Generating bar charts for gender vs. total sales amount.
      • Visualizing age group distribution.
      • Analyzing total sales amount by age group.
      • Examining the total number of orders and sales amount from the top 10 states.
      • Analyzing sales data by marital status and gender.
      • Visualizing sales data by occupation.
      • Exploring sales data by product categories and product IDs.
      • Creating a bar chart for the top 10 most sold products.

    Content used

    1. -> Data Import:
      • The code utilizes the pandas library to load and manipulate data from a CSV file.

    2. -> Data Cleaning:
      • Various data cleaning techniques, including dropping columns and handling missing values, are applied to prepare the data for analysis.

    3. -> Data Exploration:
      • The describe() method is used to generate summary statistics for the dataset, providing insights into the data's distribution.

    4. -> Data Visualization:
      • : Seaborn and Matplotlib libraries are used for data visualization, allowing for the creation of various charts and graphs to represent and explore the data visually.

    5. -> Grouping and Aggregation:
      • Data is grouped and aggregated using pandas' groupby() function to calculate total sales and orders for different categories.

    6. -> Bar Charts:
      • Bar charts are employed to visualize categorical data, making it easier to compare and analyze different categories.

    7. -> In summary, this code demonstrates a typical data analysis project workflow, including data import, cleaning, exploration, and visualization. It provides insights into various aspects of sales data, making it easier to make data-driven decisions and draw conclusions from the dataset.
