    Key Features

    1. -> User-Friendly GUI:
      • The program provides a graphical user interface (GUI) that allows users to interact with the application intuitively.

    2. -> Text Input Field:
      • Users can enter the text they want to convert to speech using a text input field within the GUI.

    3. -> Text-to-Speech Conversion:
      • When the "Speak" button is clicked, the program utilizes the pyttsx3 library to convert the entered text into speech and audibly pronounce it through the computer's speakers.

    4. -> Responsive Interface:
      • The GUI maintains responsiveness with a fixed window size, and users can't resize the application window.

    5. -> Customization Options:
      • While the program itself is relatively simple, it can be extended to include additional features such as language selection, speech rate adjustments, or voice options by leveraging the capabilities of the pyttsx3 library.
